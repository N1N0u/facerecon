نظام كامل للتعرف على الوجوه **مصنوع من الصفر باستخدام OpenCV**، باستخدام **الهاتف Android ككاميرا IP**.  
يعرض المشروع كامل **خط سير الرؤية الحاسوبية**: إنشاء مجموعة البيانات، تدريب النموذج، والتعرف في الوقت الحقيقي.

---

## 🎯 نظرة عامة

نظام التعرف على الوجه في **الوقت الفعلي** باستخدام خوارزمية **LBPH (Local Binary Patterns Histogram)**.

| المرحلة                  | الوصف                                  |
|--------------------------|----------------------------------------|
| **إنشاء dataset**        | التقاط صور الوجوه عبر IP Webcam على الهاتف |
| **تدريب النموذج**         | تدريب نموذج LBPH باستخدام الصور المجموعة |
| **التعرف في الوقت الحقيقي** | كشف وتحديد الوجوه من الفيديو المباشر   |

Pipeline:  
Camera Stream → كشف الوجوه → استخراج المميزات → Classifieur LBPH → التنبؤ بالهوية

## 🚀 التقنيات

| المكون                | التقنية                   |
|-----------------------|---------------------------|
| كشف الوجوه            | Haar Cascade Classifier   |
| استخراج المميزات      | LBP (Local Binary Pattern)|
| التعرف على الوجوه     | LBPH Classifier           |
| مصدر الفيديو          | Android IP Webcam         |
| معالجة الصور          | OpenCV + NumPy            |

---

## 🧠 الخوارزمية

يستخدم النظام **LBPH**:

LBP(x, y) = Σ s(gi - gc) * 2^i

| الرمز | المعنى           |
|-------|-----------------|
| `gc`  | البكسل المركزي  |
| `gi`  | البكسل المجاور  |
| `s(x)`| دالة العتبة     |

يقوم LBPH بإنشاء **هستوجرامات** لهذه الأنماط ويقارنها بين الوجوه.

## 📸 جمع البيانات


python face_dataset.py
- الاتصال بكاميرا الهاتف  
- كشف الوجوه  
- التقاط **30 عينة لكل مستخدم**  
- حفظ الصور في `dataset/`

تنسيق الملفات:
User.<ID>.<NumSample>.jpg

مثال:  
User.1.1.jpg
User.1.2.jpg
User.1.3.jpg

---

## 🧠 تدريب النموذج

python training.py

- تحميل الصور من `dataset/`  
- استخراج المعرفات  
- كشف الوجوه  
- تدريب نموذج **LBPH**  
- حفظ النموذج في: `trainer/trainer.yml`

---

## 🎥 التعرف على الوجه في الوقت الحقيقي

python face_recognition.py

- تشغيل الكاميرا  
- كشف الوجوه  
- التنبؤ بالهوية  
- عرض النتائج

مثال:  
User1 89%
User2 92%
Unknown
